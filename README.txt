Code Repository for Undergraduate Electrical Engineering Thesis Project
--------------------------------------------------------------------------
Thesis Title: Embedded Control System for an Autonomous Planetary Rover
Author: Aaron Lucas
Institution: University of New South Wales
Year of Submission: 2020

This Repository contains all code written for this project, including:
- A C implementation of a PID controller written for the TM4C123GH6PM
  microcontroller
- A Verilog implementation of a PID controller for use on FPGAs
- MATLAB and Simulink files for system modelling, verification and analysis
- Arduino scripts for taking experimental measurements
- Python scripts for miscellaneous purposes
